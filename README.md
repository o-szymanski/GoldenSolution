## Run locally

Download

[Docker desktop]() \
[Node.js]()

Clone the project

```bash
git clone https://github.com/o-szymanski/GoldenSolution.git
```

Go to the project directory

```bash
cd Docker
```

Start Containers

```bash
docker-compose up -d / docker-compose up --build
```
